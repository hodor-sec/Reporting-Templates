Creating malicious JAR


Create files
weblogic/jndi/Environment.java
META-INF/MANIFEST.MF

Compile file to .class
$ javac weblogic/jndi/Environment.java

Optionally test file
$ java weblogic.jndi.Environment

Create JAR file
$ jar cmvf META-INF/MANIFEST.MF weblogic.jar weblogic/jndi/Environment.class

Run JAR file
$ java -jar weblogic.jar
