Conversions to number


When a string is evaluated in a numeric context, the resulting value and type are determined as follows:
If the string does not contain any of the characters ‘.’, ‘e’, or ‘E’ and the numeric value fits into integer type limits (as defined by PHP_INT_MAX), the string will be
evaluated as an integer. In all other cases it will be evaluated as a float.

The value is given by the initial portion of the string. If the string starts with valid numeric data, this will be the value used. Otherwise, the value will be 0 (zero). Valid
numeric data is an optional sign, followed by one or more digits (optionally containing a decimal point), followed by an optional exponent. The exponent is an ‘e’
or ‘E’ followed by one or more digits.

PHP 5.6

student@atutor:~$ php -a
Interactive mode enabled
php > var_dump('0xAAAA' == '43690');
bool(true)
php > var_dump('0xAAAA' == 43690);
bool(true)
php > var_dump(0xAAAA == 43690);
bool(true)
php > var_dump('0xAAAA' == '43691');
bool(false)
