PostgreSQL


In postgresql.conf (often in {DIR}/pgsql/data/{DBNAME}/postgresql.conf)
log_statement = 'all'

Tail log
- Open Powershell as admin
- Go to logging directory
- Sort files last written:
→ dir | sort LastWriteTime | select -last 1
- Tail selected file
→ Get-Content .\postgresql_8.log -wait -tail 1
→ Get-Content .\postgresql_8.log -wait -tail 1 | Select-String -Pattern "select version"
→ Get-Content .\postgresql_8.log -wait -tail 100 | Select-String -Pattern "(somestring( >|\))|syntax error)"
→ Get-Content .\postgresql_8.log -wait -tail 100 | Select-String -Pattern error -context 0,2
