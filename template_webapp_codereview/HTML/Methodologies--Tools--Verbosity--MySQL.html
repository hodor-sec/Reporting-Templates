MySQL


Modify logging
Open the MySQL server configuration file located at /etc/mysql/my.cnf and uncomment the following lines under the Logging and Replication section:
$ sudo nano /etc/mysql/my.cnf
[mysqld]
...
general_log_file = /var/log/mysql/mysql.log
general_log = 1

Restart
After modifying the configuration file, we need to restart the MySQL server in order for the change to take effect:
$ sudo systemctl restart mysql
